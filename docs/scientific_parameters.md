# Scientific Parameters

Contained scientific resources are under `resources/`:

- `resources/cg_mdp/`
- `resources/aa_mdp/`
- `resources/forcefields/`
- `resources/mappings/`
- `resources/templates/`
- `resources/ligand_params/`
- `resources/vendor/`

Provenance and checksums are recorded in `resources/RESOURCE_MANIFEST.tsv` and
`docs/third_party_inventory.tsv`. Restricted force fields, ligand parameters,
and private regression fixtures are external user-supplied resources.
