# Martini Citations

MembraneForger retains byte-identical Martini 3 parameter files from the
official Apache-2.0 `marrink-lab/martini-forcefields` repository at commit
`784591ebdc91d762ed4df986c4650546c938f776`.

When using these files, cite the Martini 3 force-field publication and any
molecule-specific publications required by the upstream Martini documentation.
