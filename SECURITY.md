# Security

MembraneForger is a research workflow and should be run in an environment appropriate for untrusted molecular input files.

Report security concerns privately to the maintainers before public disclosure. Do not include private structures, credentials, licensed dependency files, or unpublished simulation data in reports.

The workflow does not require secrets for normal operation. If licensed tools such as Rosetta or PyRosetta are used, expose them through local environment variables rather than committing paths, licenses, or installation archives.
